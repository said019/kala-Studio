# Kala Barre Studio Design Context

## Palette

- Background: `#FFF7F2`, warm white with a blush tint.
- Foreground: `#2E201C`, warm espresso.
- Primary: `#76214D`, berry wine.
- Coral: `#E9745F`, active warmth.
- Olive: `#778455`, grounded studio tone.
- Orange: `#F58A24`, reward and energy signal.
- Blush: `#FCE6E1`, soft supporting surface.

## Typography

- Display and headings: Bricolage Grotesque, heavy weights.
- Body: Alilato with Jost fallback.
- Numbers in progress components should use tabular styling where possible.

## Layout

- Use image-led sections and asymmetry for landing-page storytelling.
- Avoid nested cards. Use framed components only for concrete tools like wallet/progress modules.
- Public pages should stay warm, bright and spacious.

## Components

- Progress rings use three meanings: Constancia, Esfuerzo, Conexion.
- Primary actions are rounded pill buttons with icons.
- Use lucide-react icons already present in the project.

## Motion

- Prefer CSS transforms and opacity.
- Keep perpetual movement subtle. Progress should feel alive but not noisy.
